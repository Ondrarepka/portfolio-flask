Before printing: in the print dialog set Scale = 100% (turn OFF “Fit to page”) and margins to None / Default, or the 25.5 mm size will be wrong. After printing, measure the ruler strip at the top — the marked 100 mm should measure exactly 100 mm. If it doesn't, divide 100 by what you measured and nudge the disc Ø, then reprint.
